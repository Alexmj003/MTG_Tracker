These files are excel files containing tables that contain card values for the set of Throne Of Eldraine from the 
TCG card game, Magic, The Gathering by Wizards of the Coast. 

The Name Columnn contains the unique identifiers for the card

The card Id values are empty, the scraping process is not outputting the correct data so the default is to not display it

The rarity column name contain the rarity value for the card. Rarity values are as followed. 
M = Mythic Rare
R = Rare
U = Uncommen
C = Common
T = Token

Listed Market Value is the Listed Market price on the date of extraction
Buylist Price is Buylist Price on the date of the extraction
Listed Median is the Listed Median price on the date of extraction
